# System Architecture - Python-Cookbook

## Project Purpose
A structured learning repository for Python practicals, following a "cookbook" style where each concept is isolated into its own script.

## Directory Structure
- `01_Basics/`: Core Python fundamentals.
  - `hello_world.py`: Basic output.
  - `print_numbers.py`: Loops and control flow.
  - `variables_and_math.py`: Data manipulation and arithmetic.
  - `string_types.py`: Different string declaration methods.

## Standards
- Use f-strings for all print formatting.
- Include comments explaining the "why" behind specific lines.
- Organize files into numbered modules for progressive learning.
