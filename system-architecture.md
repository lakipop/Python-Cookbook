# System Architecture - Python-Cookbook

## Project Purpose
A structured learning repository for Python practicals, following a "cookbook" style where each concept is mapped clearly to an execution.

## Directory Structure
- The root directory hosts individual `.py` files corresponding to different lab sheets (e.g., `practical_01.py`).
- Each file behaves like an interactive Jupyter notebook via `# %%` cell markers for ease of review.

## Standards
- Use `# %%` at the top of logical sections or specific exercises.
- Embed the question directly as a comment or docstring to maintain context.
- Include a very brief, helpful teaching note discussing the Python feature or logic directly above the solution code.
- For exercises implying multi-file imports (like constants.py), consolidate the required external module logic into the same file's cell to preserve the single-file paradigm.
